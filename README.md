# SugarWise — Comprehensive Diabetes Management Companion

A free, open-source educational website covering 65+ diabetes management topics with an India-specific focus.

**Live site:** https://aravindcleaerr.github.io/diabetes-guide/

## What's Inside

- **65+ topic pages** — from understanding diabetes to daily management, complications, diet, exercise, mental health, and more
- **Interactive features** — daily checklist, blood sugar gauges, search, animated illustrations
- **India-focused resources** — Indian diet plans, Bangalore-specific resources, financial assistance programs
- **Mobile-friendly** — responsive design that works on all devices
